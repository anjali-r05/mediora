## 🚀 Overview

**Mediora** is an AI-powered Hospital Management System designed to modernize healthcare by combining intelligent automation, operational insights, and centralized hospital management into a single platform.

Built using **Python, Flask, SQLAlchemy, and Google Gemini AI**, Mediora helps hospitals streamline workflows, improve operational efficiency, and support better patient care.

## 🛠 Tech Stack

- **Backend:** Python, Flask
- **Database:** SQLite, SQLAlchemy
- **AI:** Google Gemini API
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Charts:** Chart.js
- **Authentication:** Flask-Login
- **Deployment:** Render
- **Version Control:** Git & GitHub
